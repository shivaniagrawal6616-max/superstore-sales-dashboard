**#Super Store Sales Dashboard 📊**

Project Overview

This Power BI project analyzes Super Store sales data and provides interactive business insights through data visualization and forecasting techniques.

The dashboard helps track sales performance, profit trends, customer segments, shipping modes, and category-wise analysis. It also includes a 15-day sales forecast to predict future sales trends based on historical data.

Tools Used

- Power BI
- Excel / CSV
- DAX
- Power Query

Dashboard Features

- KPI Cards for Sales, Profit, Quantity, and Average Delivery
- Sales Analysis by Category and Sub-Category
- Sales by Segment and Payment Mode
- Region and State-wise Sales Analysis
- Monthly Sales and Profit Trend (Year-over-Year)
- Interactive Filters and Slicers
- 15-Day Sales Forecast

Forecast Analysis (Page 2)

The second page of the dashboard focuses on sales forecasting.

Using historical sales data, Power BI forecasting tools were used to predict upcoming sales trends for the next 15 days.

Forecast Insights

- Identified expected future sales patterns
- Observed fluctuations in sales trends over time
- Compared state-wise sales performance
- Forecast helps support better business planning and decision-making

Key Insights

- California and New York generated high sales compared to other states
- Technology and Office Supplies contributed significantly to sales
- Online and COD payment modes had strong usage
- Monthly sales trends showed noticeable seasonal fluctuations
- Forecast analysis suggests continued sales activity in upcoming days
